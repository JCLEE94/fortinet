Configuration Database Methods


Get method (auxiliary table)

Retrieve a list of objects using the the auxiliary command.

Request:
{
   "method": "get",
   "params": [ {
      "url": "...",
      "data": { ... }
   } ],
   "session": "...",
   "id": 1
}
Result:
{
   "method": "get",
   "result": {
      "url": "...",
      "data": [ { ... } ],
      "status": { "code": 0, "message": "..." }
   },
   "id": 1
}

 

Parameter and Returned ValuesDescriptions
urlstring
Path to the auxiliary command, refer to the command list.
data(optional) object
Upon success, return an array of objects.
statusresult object
Error code and message for the command. Refer to the error code list for all possible return values.

Exec method (auxiliary)

Execute an auxiliary command.

Request:
{
   "method": "exec",
   "params": [ {
      "url": "...",
      "data": { ... }
   } ],
   "session": "...",
   "id": 1
}
Result:
{
   "method": "exec",
   "result": {
      "url": "...",
      "data": { ... },
      "status": { "code": 0, "message": "..." }
   },
   "id": 1
}

 

Parameter and Returned ValuesDescriptions
urlstring
Path to the auxiliary command, refer to the command list.
data(optional) object
In Request:
Parameters for the command for the execution, if any. Refer to object reference for more information.
In Response:
Returned value from the execution, if any. Refer to object reference for more information.
statusresult object
Error code and message for the command. Refer to the error code list for all possible return values.
