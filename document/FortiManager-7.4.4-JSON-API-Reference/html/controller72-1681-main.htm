ADOM Controller Config (For v7.2)


This documentation applies to v7.2 devices or ADOM, latest release (build 1681).

This section includes FortiAP, FortiSwitch, and FortiExtender controller objects that are under device/vdom scope, as well as shared adom objects that are only used by controller objects. For ADOMs with per-device AP or FortiSwitch management, please refer to the Device Level documentation for API reference.
