script/execute (command)

Run script.

Supported methods: exec

Object URL:
Command object/dvmdb/script/execute
/dvmdb/global/script/execute
/dvmdb/adom/<adom_name>/script/execute
Attribute ListDescriptions
adomdatasource, refer to adom object (maximum 1) (36 bytes)
packagedatasource, refer to policy package object (maximum 1) (36 bytes)
pblockdatasource, refer to policy block object (maximum 1) (36 bytes)
scopescope object, refer to device object, or group object
Can have both device member and device group member.
scriptstring
Script name.
Response DataDescriptions
taskdatasource, refer to task (maximum 1)

script/log/list (table)

Get the of script logs of the ADOM. The amount of data to retrieve can be controlled by the range option.

This API call return the full script run result; however, retrieve all result may be slow and result in a large response. It is suggested to retrieve the summary list first to determine the proper range.

Supported methods: get

Object URL:
Table/dvmdb/global/script/log/list
/dvmdb/adom/<adom_name>/script/log/list
Attribute ListDescriptions
contentstring
The full content of the script result log.
exec_timestring
log_idint32
script_namestring
seqint32, master key
Log sequence, with 1 being assigned to the latest result.
