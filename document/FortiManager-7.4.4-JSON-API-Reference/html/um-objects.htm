image/upgrade (command)

The older API for updating the firmware of specific device. Kept for compatibility purposes.

Supported methods: exec

Object URL:
Command object/um/image/upgrade
Attribute ListDescriptions
adomdatasource, refer to adom object (maximum 1) (36 bytes)
create_taskstring
devicescope object, refer to device object, or group object
flagsoption
"f_boot_alt_partition"
"f_skip_retrieve"
"f_skip_multi_steps"
"f_skip_fortiguard_img"
"f_preview"
imageobject
buildstring
idarray
modelstring
releasestring
schedule_timestring
