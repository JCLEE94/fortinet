Device Level Config (For v7.2)


This documentation applies to v7.2 devices or ADOM, latest release (build 1681).

This section contains a selected subset of device-level configurations. For objects that are supported in ADOM, policy package, and/or templates, please refer to the respective sections.

Please note that the objects listed in this section are for general guidance only, as the availability of each object and attribute may be different depending on device platform and version. Please refer to the FOS CLI guide for more information.

JSON requests to access or modify objects at the device level can be done by a linear translation of FGT CLI object paths. An example at the end of the Configuration Database section in documentation shows how FGT CLI translation works.
