exec/fgfm/reclaim-dev-tunnel (command)

Reclaim management tunnel to device.

Request without device name specified will reclaim tunnels for all managed devices.

Supported methods: exec

Object URL:
Command object/cli/global/exec/fgfm/reclaim-dev-tunnel
/cli/global/exec/fgfm/reclaim-dev-tunnel/<device_name>
