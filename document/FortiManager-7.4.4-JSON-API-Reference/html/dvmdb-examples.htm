Move a device to another ADOM, add, set, or update the object member of the new ADOM. Delete directly from the old ADOM will result in an error.

Request:
{
    "id": 4,
    "method": "add",
    "params": [ 
        {
            "url": "/dvmdb/adom/new_adom/object member",
            "data": [ 
                {
                    "name": "fg1",
                    "vdom": "root" 
                },
                {
                    "name": "fg1",
                    "vdom": "vd1" 
                } 
            ] 
        } 
    ] 
}

Response:
{
    "id": 4,
    "result": [ 
        {
            "status": {
                "code": 0,
                "message": "OK" 
            },
            "url": "/dvmdb/adom/new_adom/object member" 
        } 
    ] 
}

ADOM revision examples


Get the checksum of root ADOM.

Request:
{
    "id": 1,
    "method": "get",
    "params": [ 
        {
            "url": "/dvmdb/adom/root",
            "option": "chksum" 
        } 
    ] 
}

Response:
{
    "id": 1,
    "result": [ 
        {
            "data": 1395275786,
            "url": "/dvmdb/adom/root" 
        } 
    ] 
}

Create a new revision of root ADOM.

Request:
{
    "id": 2,
    "method": "add",
    "params": [ 
        {
            "url": "/dvmdb/adom/root/revision",
            "data": {
                "created_by": "admin",
                "desc": "",
                "locked": 0,
                "name": "rev2" 
            } 
        } 
    ] 
}

Response:
{
    "id": 2,
    "result": [ 
        {
            "data": {
                "version": 2 
            },
            "status": {
                "code": 0,
                "message": "OK" 
            },
            "url": "/dvmdb/adom/root/revision" 
        } 
    ] 
}

Get a list of revisions in root ADOM, sorted from latest to oldest.

Request:
{
    "id": 3,
    "method": "get",
    "params": [ 
        {
            "url": "/dvmdb/adom/root/revision",
            "sortings": [

                    "version": -1

            ] 
        } 
    ]

    "id": 3,

        {
            "data": [

                    "desc": "",
                    "locked": 0,

                    "version": 2 
                },
                {
                    "created_by": "admin",
                    "created_time": 1395274727,
                    "desc": "",
                    "locked": 0,
                    "name": "rev",
                    "version": 1 
                } 
            ],
            "url": "/dvmdb/adom/root/revision" 
        } 
    ] 
}

        {

                "desc": "",
                "locked": 0,
                "name": "restored-rev" 
            } 
        } 
    ] 
}

Response:
{
    "id": 5,
    "result": [ 
        {
            "data": {
                "version": 3 
            },
            "status": {
                "code": 0,
                "message": "OK" 
            },
            "url": "/dvmdb/adom/root/revision/1" 
        } 
    ] 
}
