FortiManager 7.4.4 (2550) JSON API Reference


This document provides both query and response of FortiManager JSON API.

JSON-RPC Request

FortiManager uses the JSON-RPC protocol for communication. To use the API, send JSON query as HTTPS POST request to the URL: 
https://<ip>/jsonrpc
The general format for the JSON-RPC request query is: 
{
    "method": "...",
    "params": [ ... ],
    "session": "...",
    "id": 1
}

PropertiesDescriptions
method Supports get, add, set, update, delete, move, clone, replace, and execute. Refer to the reference for individual module for more information about availability, parameters, and response format.
params Refer to the reference for individual module for the list of parameters available for each method.
session The cookie of an active session on FortiManager. FortiManager requires a valid session in order to perform an operation via API. Refer to sys/login/user and sys/logout for creating a session using JSON API.
id An integer value which will be matched in the response data.
