_package/status (table)

Get device package/template status, or global package assign status. Optionally specify device and vdom name in the url to filter response by device.

Supported methods: get

Object URL:
Table/pm/config/global/_package/status/
/pm/config/adom/<adom_name>/_package/status
/pm/config/adom/<adom_name>/_package/status/<device_name>/<vdom_name>
Attribute ListDescriptions
stypeoption
Subtype for "template" package type. If not specified, status for all subtypes are fetched.
"_ipsec"
"_router_static"
"_cst"
"_sdwan_overlay"
"_ips_setting"
"_fap_setting"
"router_bgp"
"gl_ips_sensor"
typeoption
Package or template type. Assume "policy package" if not specified.
"policy"[Default value]
"devprof"
"ecprof"
"crprof"
"cert"
"wtp"
"wanprof"
"fsp"
"fext"
"template"
"fwmprof"
"tmplgrp"
"fab-auth"
"sase"
Response DataDescriptions
adomstring
Adom name for global assignment result.
devstring
Device name for device package/template assignment result.
pkgstring
Package or template path and name.
statusstring
Status string for the package or template assignment.
stypeoption
Subtype for "template" package type.
"_ipsec"
"_router_static"
"_cst"
"_sdwan_overlay"
"_ips_setting"
"_fap_setting"
"router_bgp"
"gl_ips_sensor"
typeoption
Package or template type. If attribute not returned in the response, the type would be "policy package".
"policy"[Default value]
"devprof"
"ecprof"
"crprof"
"cert"
"wtp"
"wanprof"
"fsp"
"fext"
"template"
"fwmprof"
"tmplgrp"
"fab-auth"
"sase"
vdomstring
Device vdom name for device package/template assignment result.
