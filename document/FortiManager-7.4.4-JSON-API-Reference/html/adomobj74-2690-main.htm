ADOM Level Objects (For v7.4)


This documentation applies to v7.4 devices or ADOM, latest release (build 2690).

This section includes all common Global and ADOM-level objects that are not part of package or template configuration. These objects will be copied down to device level during install only if they are being used by package or template. All object have prefix /pm/config/global/obj/ or /pm/config/adom/obj/ for Global or ADOM level.
