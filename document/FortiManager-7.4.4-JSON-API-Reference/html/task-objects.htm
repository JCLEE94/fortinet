task/line (table)

Read-only table.

The lines of a task can be unique identify by a combination of "name" and "vdom" attributes. As a result, there could be more than one object with the matching name, and only one of the objects will be returned if name is specified in the URL. It is recommended to apply filter on the table instead.

Supported methods: get (table), get (object)

Object URL:
